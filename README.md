# Loruki

Loruki is a responsive cloud hosting website built with HTML and CSS. It presents a clean marketing website for a fictional cloud platform, including a homepage, features page, documentation page, responsive layouts, animated sections, and reusable visual assets.

## Live Demo

[View the live site](https://lorukiiii.netlify.app)

## Features

- Responsive multi-page website
- Homepage hero section with demo request form
- Platform statistics section
- CLI feature cards
- Cloud hosting promotional section
- Supported programming language cards
- Features overview page
- Documentation page with installation commands
- Mobile and tablet layouts with media queries
- CSS animations and hover effects

## Tech Stack

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media-Queries
- CSS Animations
- Font Awesome
- Google Fonts
